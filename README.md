# 💰 Personal Finance Behavioral Analysis & ETL Pipeline

## 🎯 Project Overview
This project simulates a complete data engineering and analysis pipeline. I generated a full year of mock financial transaction data, loaded it into an SQLite database (ETL process), and used Python to uncover hidden consumer spending habits.

## 🛠️ Tech Stack
* **Language:** Python
* **Environment:** Jupyter Notebook / Google Colab
* **Database:** SQLite & SQL
* **Data Manipulation:** Pandas
* **Data Visualization:** Seaborn, Matplotlib

## 🔍 Key Findings
Through SQL querying and data visualization, I identified a distinct behavioral spending pattern:
* **Weekend Splurging:** The average transaction amount for categories like 'Entertainment' and 'Dining Out' increases significantly on weekends compared to weekdays. 
* This highlights the power of time-series analysis in predicting consumer cash flow.

## 🚀 How to View & Run
1. Open the `Finance_Data_Analysis_ipyn.ipynb` file in this repository.
2. You can view all the code, SQL queries, and generated charts directly on GitHub.
3. To run it yourself, simply download the notebook or open it in Google Colab and run the cells from top to bottom!
